Python for data analysis

Drug consumption quantified dataset

This project is about analysing the drug consumption dataset.
The problem we wanted to solve is the seven-class classification problem for each drug.

List of all drugs of the dataset

  • alcohol
  • amphet
  • amyl
  • benzos
  • caffeine
  • cannabis
  • chocolate
  • coke
  • crack
  • heroin
  • ketamine
  • legal highs
  • LSD
  • magic mushrooms
  • meth
  • nicotine
  • semer
  • vsa

Possible classes and their meaning

  • CL0 : Never Used
  • CL1 : Used over a Decade Ago
  • CL2 : Used in Last Decade
  • CL3 : Used in Last Year
  • CL4 : Used in Last Month
  • CL5 : Used in Last Week
  • CL6 : Used in Last Day

How does it work ?

In the request.py file, you can give a person (with values specified in the dataset's website) : his age, gender, education level, country, ethnicity, personnality scores values (Nscore, Escore, Oscore, Ascore, Cscore), impulsiveness and sensation seeking.
Then, run python request.py
